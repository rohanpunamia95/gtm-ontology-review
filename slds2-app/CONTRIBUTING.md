# Contributing

This project is **source-available**. The code is publicly visible for transparency, reference, and forking, but does not accept external contributions (issues, pull requests, or feature requests).

All development is done internally by the Salesforce UX team.

## Forking

You're welcome to fork this repository for your own use under the terms of the [LICENSE](LICENSE.txt).

## Reporting Security Issues

If you discover a security vulnerability, please report it through Salesforce's responsible disclosure process rather than opening a public issue.
